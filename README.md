# AutoGrader
automatically compile and checking answer all codes in the same directory

## Path
all codes have to lacate in the same directory with grade.sh

* /directory
  * **grade.sh**
  * **input.txt**
  * **answer.txt**
  * a.c
  * b.java
  * c.cpp
  * d.py
  * ...

## Usage
```md
./grade.sh
```

Or, getting more clear result:

```md
./gread.sh > result.txt
```
